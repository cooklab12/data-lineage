Portfolio Holdings

This table contains detailed information about all investment holdings across portfolios, including security identifiers, quantities, values, and other relevant attributes. Used for portfolio analysis, risk management, and regulatory reporting.

Owner: Investment Operations Team
Data Domain: Investments
Update Frequency: Daily
Retention Period: 7 years
Sensitive Data: Yes
GDPR Classification: PII Present
Column Name Data Type Critical Description Rules Actions
portfolio_id VARCHAR(20) Yes Unique identifier for the portfolio Not Null Format: P-[0-9]{8}
security_id VARCHAR(20) Yes Identifier for the security/asset Not Null Valid in Security Master
holding_date DATE Yes Date of the holding record Not Null Within past 5 years
quantity DECIMAL(20,6) Yes Number of units held Not Null >= 0
market_value DECIMAL(20,2) Yes Market value in base currency Not Null >= 0 Within 3 std deviations
cost_basis DECIMAL(20,2) No Original cost of the position >= 0
last_updated TIMESTAMP No When the record was last updated Not Null
